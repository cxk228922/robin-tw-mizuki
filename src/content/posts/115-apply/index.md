# 前後經過
這要提到我的生涯經歷了，敝人是在高一進去大安高工資訊科後才開始學習程式語言，那時候我們班導上一屆帶出一位靠著TOI上台大資工的學長，那時候就覺得他好厲害，我也好想特選。之後就是高二時開始接觸資安，二上覺得自己屌逼逼可以特選，二下了解自己真的很爛就乖乖讀書了。

之後就是高三，那時候其實我已經幾乎放棄特殊選材了，尤其是在前兩間連書審都沒過的時候，之後我也沒想到會備上交大資工。

## 自我介紹
自我介紹是一分鐘，記得**不能帶三折頁、簡報...**，全憑你的口風

這是我的事前擬稿:
```
各位教授好，我是來自市立大安高工資訊科的江寬泓，我在資安中主要研究逆向工程與數位鑑識，並且為ICEDTEA戰隊核心成員。現在主要研究windows惡意程式分析並準備金盾決賽。

我在高中時期已累積豐富的實作與領導經驗，包括帶領學校資安社團、參與戰隊，以及執行多項校內外專題如 AIS3、實習課程與畢業專題，並在其中取得優異名次。這些經歷使我具備良好的團隊合作、領導與溝通能力。此外，由於就讀技術型高中，我也有比一般高中生更紮實的硬體相關基礎，例如數位電子與邏輯電路等。

交大擁有深厚的計算機歷史，未來想透過大學課程深入學習編譯器與作業系統底層相關知識
```

## 問詢
他具體問我什麼我忘了，但他全問我逆向相關的問題

像是他給一張投影片，裡面是Ghidra主頁面，他問我"藍色的字是什麼"

或是問我我覺得現在還有閱讀machine code / assembly的必要嗎

之後就變成我在聊天，像是跟他聊要怎麼欺騙過IDA之類的

總之問題都怪怪的，然後我還一直插嘴LoL。
